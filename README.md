## 작동 방법
- App.py를 실행하여 작동합니다.
- 첫 실행시, [학년, 반, 최소화 여부]를 입력합니다.
- 학년, 반은 범위 내의 정수를, 최소화 여부는 true 또는 false를 입력합니다.
- 최소화 체크로 한눈에 다음 일정 확인이 가능합니다.

## 작동되지 않을 때
- 입력 도중 App.py가 종료된 경우, setting.txt를 제거합니다.
- 시간이 다른 경우, Win + i 를 눌러 설정에서 날짜 및 시간을 변경합니다.
